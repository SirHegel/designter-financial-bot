# 🚀 Designter Financial Bot (Python + Google Sheets API)

Este proyecto es un asistente inteligente de Telegram diseñado para automatizar la gestión financiera de mi agencia, **Designter**. Permite separar ingresos personales de empresariales mediante procesamiento de lenguaje natural.

## 📊 ¿Qué resuelve este Bot?
Como estudiante de Economía y Desarrollador, identifiqué la necesidad de tener un control de caja inmediato sin depender de hojas de cálculo manuales. 

* **Categorización Inteligente:** Identifica montos y etiquetas (Jhon vs Designter) usando RegEx.
* **Alertas de Salud Financiera:** Notificaciones automáticas cuando los gastos alcanzan el 30%, 50%, 80% o 100% de los ingresos.
* **Integridad de Datos:** Conexión directa con Google Sheets API para almacenamiento en la nube.
* **Gestión Mensual:** Creación dinámica de pestañas por mes (ej. 03-2026).

## 🛠️ Tecnologías
* **Lenguaje:** Python 3.x
* **APIs:** Telegram Bot API, Google Drive & Sheets API.
* **Librerías:** `pyTelegramBotAPI`, `gspread`, `oauth2client`.
